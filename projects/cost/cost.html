PowerCal

PowerCal was a school project to teach us how to create a secure Smart Home system. We used emulated raspberry pi modules to recreate the IOT device, MQTT to broadcast to database and Python server to run the Website to display the data retreived. When changes are made to the devices, a message notification will be sent to the users telegram chat. The telegram chat is act as a way to receive a notification from the system in replacement of an actual app, with no doubt a self relying system is best for such cases. But due to time constrains, telegram bot was our only options.
